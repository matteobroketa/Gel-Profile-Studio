Calibration model
After you click the ladder lane, the app detects intensity peaks and fits log10(bp) against migration distance. Automatic matching is order-based and should be reviewed; saturated, faint, curved, or missing ladder bands can reduce accuracy.

How to use & scientific notes

Workflow

  1. Open or drop a gel image. Rotate if the lanes are not vertical.
  2. Click the center of each lane. A narrow vertical ROI is averaged across the selected lane width; drag a line to reposition it.
  3. Name each lane in the profile panel. The label appears along the lane on the image.
  4. Choose bright- or dark-band polarity, background correction and smoothing. Use the normalization toggle for per-lane comparison, whole-image comparison, or raw grayscale signal.
  5. Optional: place a ladder lane, choose a known ladder, and review the detected peak count and calibration quality before interpreting sizes.
  6. Export a standalone HTML report, CSV profile table, annotated PNG, or project JSON.

What the profile represents

Each trace is the mean grayscale signal across a narrow lane-width strip at each vertical position. A strip is intentionally used instead of a single-pixel column because it is more robust to sensor noise, small lane curvature and pixel-level artifacts. Local-flank background correction estimates background from strips immediately outside the lane at the same migration position.

Interpretation cautions

Densitometry is semi-quantitative unless acquisition is unsaturated and calibrated. Do not compare absolute band quantity across different gels or imaging settings without appropriate standards and a validated workflow.
  • Check saturation: clipped bright bands lose quantitative information. This app flags strongly saturated lanes but cannot recover clipped signal.
  • Keep lane width consistent for comparisons. Background subtraction can materially change integrated intensity, especially on uneven gels.
  • DNA size calibration is approximately log-linear over useful migration ranges, not perfectly linear across every fragment size or gel condition. Review ladder matching before using calibrated sizes.
  • Band intensity depends on stain, imaging exposure, DNA conformation and local background. Treat intensity as relative unless the experiment includes an appropriate quantitative standard.

Design basis

The interface follows the compact, dark, utility-first style used across Matteo Broketa’s browser tools and GitHub profile: system typography, bordered panels, restrained controls, local-first analysis and explicit export. The analysis workflow borrows established ideas from ImageJ/Fiji gel profiling: lane ROIs, one-dimensional density traces, background handling and peak-area interpretation. Recent tools such as GelGenie show the value of automating band segmentation; this v1 deliberately keeps the core analysis transparent and inspectable rather than hiding it behind a model.
